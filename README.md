# Foodie-Wheel
# 🌀 今天吃什么？

一个用 HTML + JavaScript 实现的「随机转盘选菜网站」，让你不再纠结吃什么！

## 🎯 功能介绍

- 🍽️ 点击“开始”按钮，转盘旋转随机选出一道菜品
- 🔁 可点击左上角菜单 → 更改转盘信息（增删菜品）
- 💾 所有数据保存在浏览器本地，刷新/关闭也能保留
- 🔎 选中结果可点击，自动跳转 Google 搜索该菜做法
